Getting Started
===================

Installation
^^^^^^^^^^^^^

The cfspopcon package is available on the `Python Package Index <https://pypi.org/>`, thus installation is as simple as:

.. code::

  >>> pip install cfspopcon
  >>> radas -d ./radas_dir
